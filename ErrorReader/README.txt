Error Reader



requirements:
	
python 3.5 <



Instructions
:	
	1. open prefered terminal/cmd 
	2. cd into script directory (cd Desktop/ErrorReader)
	
	3. enter python error_reader.py -d [device_name] -e [error_number]
	
	4. rerun for all errors



Parameters
:	
	--device, -d: interger number of device, currently supported:

		'1' PolyPhase (KW), '2' PTZ, '3' TUF flow.
	
	

	--error, -e: interger value of error.
